To run the server

First, install the npm packages for the server. Navigate into server directory and in a command line terminal run:
### `npm install`

Second, run this command in terminal
### `node index`

If successful, the terminal will print 'Hackbox online on port 8080!' 


To run the client

First, install the npm packages for the server. Navigate into client directory and in a command line terminal run:
### `npm install`

Run the angular application with this command
### `ng serve

The application is now live on http://localhost:4200
